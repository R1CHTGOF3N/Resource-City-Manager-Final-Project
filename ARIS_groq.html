API Access — How to connect live data
The egov.kz API v4 provides real-time access to all datasets. A free API key is obtained from the Developer Cabinet at data.egov.kz. The endpoint pattern is:
GET https://data.egov.kz/api/v4/{dataset_uri}/v1?apiKey={YOUR_KEY}
GET https://data.egov.kz/api/v4/opendata-api-uri156/v1?apiKey={YOUR_KEY}
// Returns JSON array — Almaty citizens' appeals dataset
For this demo, data was pre-fetched and cached locally to avoid CORS restrictions. A production deployment would use a lightweight Node.js proxy to call the API server-side and feed real-time values into ARIS.
